# cpp-advanced-vector

Заголовок: аналог vector с размещением в сырой памяти

Язык: С++

Описание: тренировочный проект в рамках ЯндексПрактикума с пошаговым созданием с нуля контейнера, аналогичного vector. Прописывается класс RawMemory, позволяющий выделять и оперировать с сырой памятью. Класс Vector с настроенной итерацией и базовыми методами добавления, удаления, коирования и перемещения элементов в выделенной сырой памяти в зависимости от вместимости последней. Настроены конструкторы с вариативными шаблоны для реализации аналога быстрого Emplace.
